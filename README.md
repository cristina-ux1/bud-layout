### Instructions

``` bash
# Clone the repository: 
git clone https://github.com/cristinabalica/bud-layout.git

# Install dependencies:
npm install

# Install Parcel globally on your system by using npm: 
npm install -g parcel-bundler

# Start the web server by using the following command: 
parcel index.html

# Then open in your browser
http://localhost:1234/ 
```
